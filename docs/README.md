**기능별 목록**

[x] 다리 길이 입력 메소드  
[x] 다리 길이의 유효성 검증  
[x] 입력값이 숫자인지 확인하는 메소드  
[x] 이동할 칸 입력 메소드  
[x] 재시작/종료 입력 메소드  
[x] 현재까지 이동한 다리 출력 메소드  
[x] 최종 결과 출력 메소드  
[x] 다리를 길이에 맞춰 생성하는 메소드  
[x] 사용자가 게임을 다시 시도할 때 사용하는 메소드  
[x] 사용자가 칸을 이동할 때 사용하는 메소드

**예외처리**  
[x]다리 길이 입력이 숫자가 아닐때  
[x]다리 길이 입력이 3~20범위가 아닐 때  
[x]다리 움직임 입력이 U, D이 아닐 때  
[x]게임이 끝난 후 종료 입력이 R, Q가 아닐때
